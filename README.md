# Byte Snack - One Day One Qoutes and Facts

Website auto-generate **Qoute of the Day** and **One Day One Fact**. Build with pure HTML, CSS, and JavaScript.
<br>
You can access the website using the following link : https://quotes-api-brown.vercel.app/

---

## Features
- **Qoute of the Day** Get inspiring Qoutes from famous people
- **One Day One Fact** Get interesting facts every day
- **Copy to Clipboard** Easy copy qoutes and facts
- **Share Function** You can share those facts
- **Fully Responsive** Works perfectly on Mobile, tablets, and  Desktop
